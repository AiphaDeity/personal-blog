# 📝 个人博客

用 Python + Flask 从零搭建的个人博客网站，支持在线写文章、查看文章列表和详情。

## 🌍 在线访问

👉 **[https://aiphadeity.pythonanywhere.com](https://aiphadeity.pythonanywhere.com)**

---

## ✨ 已实现功能

- 📋 **文章列表**：首页按时间倒序展示所有文章
- 📝 **写文章**：通过网页表单在线撰写并发布
- 👀 **文章详情**：点击标题查看完整内容
- 🎨 **自定义样式**：深蓝色导航栏，白色卡片布局
- 🗃️ **SQLite 数据库**：文章永久保存，重启不丢失
- ⏰ **自动时间戳**：每篇文章自动记录发布时间
- 🌐 **免费上线**：通过 PythonAnywhere 部署，全球可访问

---

## 🛠️ 技术架构

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端框架 | Flask | 处理请求、路由分发 |
| 前端模板 | Jinja2 + HTML | 动态渲染页面 |
| 样式 | CSS | 自定义响应式样式 |
| 数据库 | SQLite | 轻量级文件数据库 |
| 部署平台 | PythonAnywhere | 免费 Python 托管 |
